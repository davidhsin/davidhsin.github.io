### Cheatsheet

```bash
brew uses --installed <package> # 列出哪些套件依賴 package

brew deps --tree --installed <package> # 列出該 package 的相依套件
```

```bash
bear -- make # 輸出 compile_commands.json，解決 .c/c++ 檔 file not found 路徑問題
```

```bash
git log --reverse --oneline # 按時間順序（最早的提交在前）顯示當前分支的提交歷史，每個提交顯示在單獨的一行上。
```

```bash
tmux display-message -p '#s' # `#S` 是 tmux 的 format 變數，代表 current session name。
