```python
class human():
  def __init__(self):  # 建立預設屬性的寫法
    self.eye = 2       # 兩個眼睛
    self.ear = 2       # 兩個耳朵
    self.nose = 1      # 一個鼻子
    self.mouth = 1     # 一張嘴巴

oxxo = human()         # 製作一個名為 oxxo 的物件
print(oxxo.eye)        # 得到 2 ( 印出 oxxo 的 eye 屬性 )。
```
